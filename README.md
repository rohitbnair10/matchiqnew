## The Problem

Searching for property in Dubai means scrolling through hundreds of listings with no real intelligence behind the filters. You know what you want — but the platform doesn't learn from you.

## What MatchIQ Does

MatchIQ is a **Chrome extension** that sits on top of [Property Finder](https://www.propertyfinder.ae) and adds a layer of AI-powered personalization:

- You describe your ideal home in natural language (location, vibe, must-haves, budget)
- GPT-4 analyzes active listings in real time and surfaces the ones that genuinely match
- No more manual scrolling — the feed gets smarter the more context you give it

---

## How it works

1. Install the Chrome extension
2. Open any Property Finder search page
3. Enter your preferences in the MatchIQ sidebar
4. GPT-4 re-ranks and highlights matching listings on the page in real time

---

## Built with

- **Extension:** JavaScript (Chrome Extension Manifest V3)
- **AI:** GPT-4 (OpenAI API) for real-time listing analysis
- **Deployment:** Vercel (backend API)
